Acerca de

Para hacer esta aplicación web, Christian y Yesid utilizarón AngularJS, Materialize y Bootstrap 4 para el diseño. También utilizarón Firebase para el manejo de los datos y el lenguaje JavaScript para la interpretación de ellos. Se descargaron algunos modulos o librerias que fueron necesarios para llevar a cabo el desarrollo, entre los modulos utilizados se encuentran: el modulo de Google Maps, el modulo de Materialize, el Ngb-Module para utilizar algunos componentes de Bootstrap 4 especialmente para Angular. Se crearon cinco componentes para la implementación de la herramienta, uno para las busquedas de hoteles, otro para los comentarios, uno que es el contenedor general, otro para crear nuevos hoteles y el ultimo para la sección sobre nosotros. Se aplicaron estilos en CSS a cada uno de los componentes, se utilizaron las fuentes de Googgle y los iconos de Fontawesome y Materialicons.
